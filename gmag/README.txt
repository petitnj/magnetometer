Use the following command to compile:
  gcc -Wall -lm GBOMag.c ini.c ini.h -o GBOMag

To call the program use:
GBOMag /path/to/ini/file
Or
GBOMag
(Second way to call looks for gbo.ini in the calling directory)

GBOMag_Test is compiled with debugging
GB0Mag_Test1 is not compiled with debugging (called by GBOMagStartup.sh)
